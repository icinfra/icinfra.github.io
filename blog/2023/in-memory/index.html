in memory

October 1, 2023

2023

母亲的影子总是让我心生温暖。然而，我不得不承认，即便是她的爱，也无法完全消解生活中遭受的所有冷漠与伤害。

母亲的牺牲

在那个贫穷的年代，我母亲身负重担。她的生活早早便陷入了艰辛之中，从小学二年级就开始背负着无尽的责任和期望。当小姨还是个婴儿时，母亲就每天带着她一起上学。学校的小朋友和老师嘲笑她，骂她“带人的又来了”，而小姨的哭闹则让母亲多次被同学们逐出教室，留在窗外偷听课程。

这些重压让我的母亲决定终止自己的学业，她停留在了小学二年级。她选择了留在家中帮助照顾家人和农田，而将寄托希望的机会留给了她的大姐和其他的兄弟姐妹。

温馨的周末

每当我回想起小学时的周末，记忆总是明媚的。母亲会带着我和姐姐去外祖母家，那里总是充满了爱和关怀。每次去外祖母家，外祖母都会留下最大的鸡腿给我，还准备好几个红鸡蛋让我们带回家。

隔壁镇的生活

然而，好日子并没有持续太久。由于外祖父母需要去城里帮大舅照看孙子，母亲被叫去外祖母家帮助种地、照看我小舅上中学。我也跟随着母亲，离开了我生长的地方，来到了隔壁镇的外祖母家。虽然只是隔了一个镇，但我却感觉与家乡相隔万里，几年都没有回去过。

艰难的日子

之后的日子更为艰难。父母外出打工，我和外祖父母生活在一起。生活在外祖父母家的那几年，我常常受到外祖母和小舅的恶言相向和精神虐待。我不明白，为什么一个无知天真的小孩，会受到如此的对待。

母亲的无奈

即便如此，母亲依旧每年春种夏收&夏种秋收请假回到娘家帮忙务农。她每个月只有三四百块钱的工资，但为了家里，她愿意付出所有。但是，当我受到外祖母和小舅的虐待时，她却无法成为我的坚实后盾。她在电话里或者回到家之后，总是会找一些外祖母报告的“坏”事情责怪我。

生活的教训

生活虽然充满了艰辛和无奈，但每一次的打击都让我更为坚强。母亲的爱虽然无法抵挡所有的苦楚，但她的故事和牺牲却成为了我人生道路上不灭的光。我从这一切中汲取力量和勇气，去面对生活的一切风风雨雨。
