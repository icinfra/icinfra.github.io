# Diagram-based Numbering
## Single Level
在UML时序图的背景处，右键选择Sequence Number -> Single Level。

![](/assets/img/Pasted%20image%2020230317150009.png)

只要在同一个图中，编号就是连续的。

![](/assets/img/Pasted%20image%2020230317150053.png)

## Nested Level
在UML时序图的背景处，右键选择Sequence Number -> Nested Level。效果如下，
